# Teaching Dossier Website

## How to use

1. Upload these files to a GitHub repository:
   - index.html
   - style.css
   - assets/

2. Put your PDFs inside:
   assets/documents/

3. Rename your PDFs to match the placeholder links, or edit the links in index.html.
